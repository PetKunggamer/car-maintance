## 🚀 วิธีเริ่มใช้งาน

### 1. เปิดเว็บ

เปิด Browser (Chrome / Edge / Firefox) แล้วพิมพ์ URL ที่ได้รับ เช่น:
```
https://maintenance-dashboard-xxx.vercel.app
```
ระบบจะโหลดพร้อม **ข้อมูลตัวอย่าง** ให้ดูก่อนเลย ไม่ต้องทำอะไรเพิ่ม

### 2. เตรียมไฟล์ Excel

> 💡 **แนะนำ:** กดปุ่ม **"ดาวน์โหลด Template"** ที่มุมขวาบน เพื่อรับไฟล์ Excel ที่มีหัวคอลัมน์ถูกต้องพร้อมกรอก

ไฟล์ต้องมีหัวคอลัมน์แถวแรกดังนี้:

| คอลัมน์ | ตัวอย่าง | หมายเหตุ |
|--------|---------|---------|
| **วันที่** | `2026-06-11` | รูปแบบ YYYY-MM-DD |
| **ทะเบียนรถ** | `กข-1234` | |
| **ประเภทรถ** | `Toyota Hilux Vigo` | |
| **เลขไมล์** | `85000` | ตัวเลข |
| **อาการ/ปัญหา** | `น้ำมันเครื่องรั่ว` | |
| **รายการซ่อม** | `เปลี่ยนซีลเครื่องยนต์` | |
| **หมวดหมู่ซ่อม** | `เครื่องยนต์` | ดูค่าที่รองรับด้านล่าง |
| **อู่/ผู้ซ่อม** | `อู่ช่างสมชาย` | |
| **ค่าอะไหล่** | `2500` | ตัวเลข (บาท) |
| **ค่าแรง** | `1500` | ตัวเลข (บาท) |
| **รวมค่าใช้จ่าย** | `4000` | ตัวเลข (บาท) |

### 3. อัปโหลดไฟล์ Excel

มี **2 วิธี:**

**วิธีที่ 1 — คลิกปุ่ม**
1. กดปุ่ม **"อัปโหลด Excel"** สีน้ำเงิน ที่มุมขวาบน
2. เลือกไฟล์ `.xlsx` จากคอมพิวเตอร์
3. รอสักครู่ — ระบบจะอัพเดตข้อมูลทุกหน้าอัตโนมัติ

**วิธีที่ 2 — ลากวาง (Drag & Drop)**
1. คลิกเมนู **"นำเข้า Excel"** ในแถบซ้าย
2. **ลากไฟล์ Excel จาก File Explorer มาวาง** ในกล่องที่แสดง
3. ระบบจะโหลดข้อมูลทันที

> ✅ เมื่ออัปโหลดสำเร็จ จะมีข้อความสีเขียวแจ้ง เช่น *"นำเข้าสำเร็จ 28 รายการ"*

---

### 4. ดูภาพรวม (Dashboard)

หน้าแรกหลังเปิดเว็บ แสดงสรุปข้อมูลทั้งหมด:

- **การ์ด 4 ใบ** — ตัวเลขสรุปสำคัญ อัพเดตอัตโนมัติเมื่ออัปโหลด Excel ใหม่
- **กราฟแท่ง** — ค่าใช้จ่ายซ่อมแต่ละเดือน ดูได้ว่าเดือนไหนสูงผิดปกติ
- **กราฟวงกลม** — สัดส่วนงานซ่อมแต่ละหมวด
- **ตารางงานค้าง** — รายการที่ยังไม่เสร็จ แสดงท้ายหน้าเสมอ

---

### 5. ค้นหาและกรองข้อมูล

คลิก **"ตารางซ่อมทั้งหมด"** ในเมนูซ้าย จะเห็นตัวกรอง 5 แบบ:

| ตัวกรอง | ใช้สำหรับ |
|--------|---------|
| **ทะเบียนรถ** | ดูรถเฉพาะคัน |
| **เดือน/ปี** | ดูเฉพาะช่วงเวลาที่ต้องการ |
| **สถานะ** | เลือกดูเฉพาะงานเสร็จ / กำลังซ่อม / รอ |
| **หมวดหมู่** | ดูงานประเภทเดียวกัน เช่น เบรก, ยาง |
| **ค้นหา** | พิมพ์อะไรก็ได้ เช่น ชื่อช่าง, อาการ |

> เลือกค่าแล้วตารางอัพเดตทันที — กดปุ่ม **"รีเซ็ต"** เพื่อล้างตัวกรองทั้งหมด

---

### 6. ดูประวัติซ่อมรายคัน

**คลิกที่ "ทะเบียนรถ" (ตัวหนังสือสีน้ำเงิน)** ในตารางหน้าใดก็ได้ จะเปิด popup แสดง:

- จำนวนครั้งที่ซ่อมทั้งหมด
- ค่าใช้จ่ายรวมของรถคันนั้น
- เลขไมล์ล่าสุด
- ตารางประวัติทุกครั้งที่เข้าซ่อม

กด **×** หรือคลิกนอกกล่อง เพื่อปิด

---

### 7. ติดตามงานค้าง

คลิก **"งานค้าง"** ในเมนูซ้าย — ตัวเลขสีแดงบอกจำนวนงานที่ยังค้างอยู่

แสดงเฉพาะรายการที่มีสถานะ **"รอดำเนินการ"** และ **"กำลังซ่อม"** พร้อมหมายเหตุที่บันทึกไว้

---

### 8. ส่งออกข้อมูลเป็น Excel

1. กดปุ่ม **"ส่งออก Excel"** สีเขียว ที่มุมขวาบน
2. ไฟล์ชื่อ `maintenance_export.xlsx` จะดาวน์โหลดไปที่โฟลเดอร์ **Downloads**

## ❓ คำถามที่พบบ่อย

<details>
<summary><b>อัปโหลด Excel แล้วขึ้น "เกิดข้อผิดพลาด" ต้องทำอย่างไร?</b></summary>

ตรวจสอบ 3 จุด:
1. ชื่อหัวคอลัมน์แถวแรก **ต้องตรงทุกตัวอักษร** กับที่กำหนด
2. ไฟล์ต้องเป็นนามสกุล `.xlsx` หรือ `.xls`
3. ลองบันทึก (Save) ไฟล์ Excel ใหม่อีกครั้ง แล้วอัปโหลดซ้ำ
</details>

<details>
<summary><b>รีเฟรชหน้าเว็บแล้วข้อมูลหาย ทำอย่างไร?</b></summary>

ข้อมูลที่อัปโหลดอยู่ในหน่วยความจำ Browser เท่านั้น เมื่อรีเฟรชหรือปิดแท็บ **ข้อมูลจะหายไป** ต้องอัปโหลดไฟล์ Excel ใหม่ทุกครั้งที่เปิดเว็บ ดังนั้น **เก็บไฟล์ Excel ต้นฉบับไว้เสมอ**
</details>

<details>
<summary><b>จะเพิ่มข้อมูลการซ่อมใหม่ได้อย่างไร?</b></summary>

ระบบนี้ไม่รองรับการกรอกข้อมูลโดยตรง ให้:
1. เปิดไฟล์ Excel ต้นฉบับ
2. เพิ่มแถวข้อมูลใหม่
3. บันทึกไฟล์
4. อัปโหลดเข้าระบบอีกครั้ง
</details>

<details>
<summary><b>วันที่แสดงผิด ต้องกรอก Excel แบบไหน?</b></summary>

กรอกวันที่รูปแบบ `YYYY-MM-DD` เช่น `2026-06-11` หรือตั้งค่า Format ใน Excel เป็น Date แบบ `YYYY-MM-DD` แนะนำใช้ปี ค.ศ. (ไม่ใช่ พ.ศ.) เพื่อความแม่นยำ
</details>

<details>
<summary><b>กราฟไม่แสดงผล หน้าจอเปล่า ทำอย่างไร?</b></summary>

กด `Ctrl + F5` (Windows) หรือ `Cmd + Shift + R` (Mac) เพื่อโหลดหน้าใหม่ทั้งหมด หากยังไม่แสดง ให้ลองใช้ **Google Chrome** หรือ **Microsoft Edge**
</details>

<details>
<summary><b>ใช้งานบนมือถือได้ไหม?</b></summary>

ได้ รองรับทั้ง iPhone และ Android แต่เนื่องจากตารางมีหลายคอลัมน์ แนะนำให้ใช้บน **คอมพิวเตอร์หรือแท็บเล็ต** เพื่อประสบการณ์ที่ดีกว่า

**บันทึกลงหน้าจอมือถือ:**
- iOS: แตะ **แชร์ (□↑)** → **Add to Home Screen**
- Android: แตะ **⋮** → **Add to Home Screen**
</details>
